# Etta

An adaptive, gamified, offline-first learning app that takes an adult learner from
foundational algebra to university-level astrophysics. Native macOS desktop app built
with Tauri 2.x, React 19, TypeScript, and Rust. All content is generated on demand by a
frontier LLM (Anthropic Claude); all state is stored locally in SQLite. There is no
backend server, no accounts, and no cloud sync.

> Milestone 0 — foundation scaffold. Features land in later milestones.
